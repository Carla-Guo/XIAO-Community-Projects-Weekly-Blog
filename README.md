# XIAO Community Projects Weekly

A curated weekly collection of the most interesting community projects built with **Seeed Studio XIAO** development boards.

---

## About This Collection

Every week, we search across GitHub, YouTube, and Instructables to surface the most innovative and creative XIAO-based projects from the maker community. Each entry includes a brief description, key technical details, and hardware recommendations — so you can quickly find inspiration for your next XIAO project.

---

## Collection Structure

Each weekly issue contains:
- **10+ curated projects** from the past 7 days
- **Project descriptions** with technical highlights and use cases
- **Hardware recommendations** — matched XIAO board + expansion + Grove modules
- **Board Selection Guide** — full XIAO series comparison table
